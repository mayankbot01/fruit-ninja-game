# Fruit Ninja Clone

A fully functional web-based clone of the classic Fruit Ninja game built using HTML5 Canvas and Vanilla JavaScript.

## 🍎 Features
- **Dynamic Slicing**: Real-time collision detection for slicing fruits with mouse or touch.
- **Particle Effects**: Juicy explosions and juice splatters when fruits are sliced.
- **Game Physics**: Gravity-based fruit tossing and rotation.
- **Scoring System**: Keep track of your high score as you slice more fruits.
- **Lives & Bombs**: 3 lives system. Avoid slicing the bombs or it's game over!
- **Responsive Design**: Playable on desktop and mobile devices.

## 🎮 How to Play
1.  **Slice**: Click and drag (or swipe) across the screen to slice fruits.
2.  **Avoid Bombs**: Slicing a bomb results in an immediate Game Over.
3.  **Don't Miss**: If you let 3 fruits fall without slicing them, the game ends.
4.  **Score**: Each fruit sliced gives you 10 points.

## 🛠️ Technology Stack
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Graphics**: HTML5 Canvas API
